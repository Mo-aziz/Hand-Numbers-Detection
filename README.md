# Hand-Numbers-Detection Project

This project runs a hand/number detection script using a Python virtual environment.

## Project Structure

```
project-folder/
│
├─ hand_detection_env/        # Python virtual environment
├─ numberdetect.py            # Main script
└─ README.md
```

## Requirements

* Windows OS
* Python 3.x installed
* Required Python packages installed inside the virtual environment

## Setup & Run

### 1️- Activate the Virtual Environment

Open **Command Prompt** or **PowerShell** inside the project directory and run:

```bash
.\hand_detection_env\Scripts\activate
```

You should see the environment name appear in your terminal, indicating it is active.

### 2️- Run the Project

After activating the environment, execute:

```bash
python numberdetect.py
```

## Troubleshooting

* **Activation not working?** Make sure you are using Windows and Python is added to PATH.
* **Module not found errors?** Install the missing package inside the activated environment.
